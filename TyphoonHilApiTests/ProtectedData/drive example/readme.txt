To see the example in action run the create_models.py script.

Files explained:
  1. drive_template.tse - template model which will serve as a basis for customization
  2. example_project.cmp - example project file that defines 2 variable positions in the template model and demonstrates different ways for specifying the options
  3. create_models.py - example Python script that generates models for all the available model configurations
  4. Additional files:
      - configs folder contains a number of configuration files (used in the script for model generation)
      - libs folder contains a library used in the project 
      - options folder contains a number of option files (referenced in the project file)
